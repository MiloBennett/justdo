# 课表个性化设置功能实现文档

## 功能概述

在原有课表功能的基础上，新增了个性化设置模块，让用户可以根据自己的喜好自定义课表的显示样式。

## 新增文件

### 1. 数据模型

**文件**: `lib/models/schedule_settings.dart`

设置数据模型包含以下配置项：

#### 显示设置
- `showWeekend`: 是否显示周末（周六、周日）
- `showTime`: 是否在节次旁显示上课时间
- `totalSections`: 每日节数（8/10/12节）
- `showCourseTeacher`: 是否在课程卡片上显示教师姓名

#### 样式设置
- `cardBorderRadius`: 课程卡片圆角大小（0-16px）
- `cardHeight`: 课程卡片高度（40-80px）
- `enableCardShadow`: 是否启用卡片阴影效果

#### 颜色设置
- `backgroundColor`: 课表背景颜色
- `gridLineColor`: 网格线颜色
- `todayHighlightColor`: 当天高亮颜色
- `cardShadowColor`: 卡片阴影颜色

#### 字体设置
- `courseNameFontSize`: 课程名称字号（10-18）
- `classroomFontSize`: 教室字号（8-14）
- `sectionNumberFontSize`: 节次编号字号
- `timeFontSize`: 时间显示字号

### 2. 设置页面

**文件**: `lib/screens/settings_screen.dart`

设置页面包含以下功能模块：

1. **显示设置区**
   - 显示周末开关
   - 显示时间开关
   - 每日节数选择（8/10/12节）
   - 显示教师开关

2. **样式设置区**
   - 卡片圆角滑块调节
   - 卡片高度滑块调节
   - 卡片阴影开关

3. **字体设置区**
   - 课程名称字号滑块调节
   - 教室字号滑块调节

4. **颜色设置区**
   - 背景颜色选择器
   - 今天高亮颜色选择器

5. **实时预览区**
   - 根据当前设置实时预览课表样式
   - 显示课程卡片效果

## 修改文件

### 1. 主页框架

**文件**: `lib/screens/main_screen.dart`

- 添加设置按钮（右上角浮动按钮）
- 将设置参数传递给课表页面
- 管理全局设置状态

### 2. 课表显示页面

**文件**: `lib/schedule_screen.dart`

- 接收设置参数
- 根据设置动态调整显示：
  - 显示/隐藏周末
  - 显示/隐藏时间
  - 调整节次数量
  - 应用颜色配置
  - 应用字体大小
  - 应用卡片样式

## 功能特点

### 个性化配置

1. **灵活的显示选项**
   - 可选择是否显示周末，适应不同学校安排
   - 可选择是否显示具体上课时间
   - 可调整每日节数（8/10/12节）

2. **丰富的样式定制**
   - 8种预设背景颜色
   - 8种高亮颜色
   - 可调节卡片圆角和高度
   - 可开关阴影效果

3. **字体大小调节**
   - 课程名称字号可调
   - 教室字号可调
   - 滑块操作，直观方便

4. **实时预览**
   - 修改设置即时看到效果
   - 预览区域模拟真实课表显示

### 用户体验

1. **操作简便**
   - 开关切换，滑块调节
   - 颜色点击选择
   - 一键保存设置

2. **即时反馈**
   - 所有修改立即生效
   - 预览区域实时更新

## 使用说明

### 进入设置

1. 点击主页右上角的设置图标（齿轮）
2. 进入课表设置页面

### 修改设置

1. **显示设置**
   - 点击开关按钮切换显示/隐藏
   - 下拉选择每日节数

2. **样式设置**
   - 拖动滑块调节圆角和高度
   - 点击开关切换阴影效果

3. **字体设置**
   - 拖动滑块调节字号大小

4. **颜色设置**
   - 点击颜色块打开颜色选择器
   - 选择喜欢的颜色

### 保存设置

1. 点击右上角"保存"按钮
2. 返回课表页面查看效果

## 技术实现

### 状态管理

- 使用 `ScheduleSettings` 类封装所有设置
- 通过 `copyWith` 方法实现设置的不可变更新
- 在页面间传递设置对象

### 页面通信

- 主页管理全局设置状态
- 通过构造函数将设置传递给课表页面
- 设置页面返回修改后的设置对象

### 动态渲染

- 课表页面根据设置动态生成 UI
- 使用条件渲染控制元素显示
- 使用变量控制样式参数
