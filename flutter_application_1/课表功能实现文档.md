# 课表功能实现文档

## 功能概述

在原有待办事项应用的基础上，新增了课表管理功能，方便大学生根据课表进行时间安排。

## 新增文件

### 1. 数据模型

**文件**: `lib/models/course.dart`

课程数据模型包含以下字段：
- `id`: 课程唯一标识
- `name`: 课程名称
- `classroom`: 教室
- `teacher`: 教师
- `dayOfWeek`: 星期几 (1-7，表示周一到周日)
- `startSection`: 开始节次
- `endSection`: 结束节次
- `startTime`: 开始时间
- `endTime`: 结束时间
- `color`: 课程颜色（用于区分不同课程）
- `startWeek`: 开始周次
- `endWeek`: 结束周次

### 2. 页面文件

**文件**: `lib/screens/add_course_screen.dart`

添加课程页面，包含以下功能：
- 输入课程名称、教室、教师
- 选择星期几（周一到周日）
- 选择开始和结束节次（1-10节）
- 设置周次范围（1-20周）
- 选择课程颜色（8种预设颜色）

**文件**: `lib/screens/schedule_screen.dart`

课表显示页面，包含以下功能：
- 显示一周的课表（周一到周日）
- 左侧显示节次和时间
- 顶部显示星期几
- 中间显示课程卡片
- 支持切换周次（1-20周）
- 点击课程卡片可删除课程
- 高亮显示当天

**文件**: `lib/screens/main_screen.dart`

主页框架，包含以下功能：
- 底部导航栏（待办、课表）
- 使用 IndexedStack 保持页面状态

## 修改文件

**文件**: `lib/main.dart`

- 修改应用标题为"大学生助手"
- 修改主页为 MainScreen

## 功能特点

### 课表功能

1. **课程管理**
   - 添加课程：输入课程详细信息
   - 删除课程：点击课程卡片确认删除

2. **时间显示**
   - 支持10节课的时间安排
   - 每节课显示具体时间
   - 课程卡片显示课程名称和教室

3. **周次管理**
   - 支持1-20周的周次切换
   - 课程按周次范围显示
   - 当前周次高亮显示

4. **视觉设计**
   - 8种预设颜色区分不同课程
   - 课程卡片圆角设计
   - 当天高亮显示

### 待办功能（原有）

- 添加待办事项
- 标记完成/未完成
- 删除待办事项
- 显示完成进度

## 使用说明

1. **添加课程**
   - 点击课表页面右下角的"+"按钮
   - 填写课程信息
   - 点击"添加课程"按钮

2. **删除课程**
   - 在课表页面点击课程卡片
   - 确认删除

3. **切换周次**
   - 点击顶部周次显示区域的左右箭头
   - 或点击右上角日历图标选择周次

4. **查看课表**
   - 底部导航栏点击"课表"
   - 查看当前周的课程安排

## 技术实现

- 使用 Flutter Material 3 设计
- 使用 StatefulWidget 管理状态
- 使用 Navigator 进行页面跳转
- 使用 Stack 和 Positioned 实现课表布局
- 使用 IndexedStack 保持页面状态
