## 详细修改说明

### 1. 课程模型 (`lib/models/course.dart`)

**新增内容：**

```dart
// 单双周类型枚举
enum WeekType {
  every, // 每周
  odd,   // 单周
  even,  // 双周
}
```

**修改内容：**
- 移除 `startTime` 和 `endTime` 字段（时间由时间表决定）
- 新增 `weekType` 字段，类型为 `WeekType`
- 新增 `shouldShowInWeek(int week)` 方法，判断指定周次是否显示该课程
- 新增 `weekTypeText` getter，获取单双周类型的显示文本
- 修改 `weekRange` getter，包含单双周信息

**核心逻辑：**
```dart
bool shouldShowInWeek(int week) {
  if (week < startWeek || week > endWeek) {
    return false;
  }
  switch (weekType) {
    case WeekType.every:
      return true;
    case WeekType.odd:
      return week % 2 == 1;
    case WeekType.even:
      return week % 2 == 0;
  }
}
```

```dart
// 节次时间类
class SectionTime {
```

```dart
List<Course> _getCoursesForDayAndWeek(int day) {
  return _courses.where((course) {
    return course.dayOfWeek == day &&
        _currentWeek >= course.startWeek &&
        _currentWeek <= course.endWeek;
  }).toList();
}

// 修改后的过滤逻辑
List<Course> _getCoursesForDayAndWeek(int day) {
  return _courses.where((course) {
    return course.dayOfWeek == day && 
           course.shouldShowInWeek(_currentWeek);
  }).toList();
}
```

## 功能说明

### 单双周课程

1. **添加课程时**
   - 选择"周次类型"：每周、单周、双周
   - 设置开始周次和结束周次

2. **显示规则**
   - 每周：在指定周次范围内每周都显示
   - 单周：只在奇数周显示（1, 3, 5...）
   - 双周：只在偶数周显示（2, 4, 6...）

3. **周次显示**
   - 顶部显示当前周次及单双周标识
   - 例：`第3周(单周)`、`第4周(双周)`

### 自定义上课时间

1. **进入设置**
   - 课表页右上角设置按钮 → 上课时间设置

2. **修改时间**
   - 点击任意节次 → 弹出编辑对话框
   - 输入开始时间和结束时间（格式：HH:MM）
   - 点击确定保存

3. **重置时间**
   - 点击"重置默认"按钮
   - 恢复为系统默认时间方案

4. **时间显示**
   - 节次旁显示对应时间
   - 课程卡片显示时间范围（卡片足够高时）

## 使用示例

### 示例1：添加单周课程

1. 点击课表页"+"按钮
2. 填写课程信息
3. 周次类型选择"单周"
4. 开始周次：1，结束周次：16
5. 点击添加

**结果：** 该课程只在第1、3、5、7、9、11、13、15周显示

### 示例2：添加双周课程

1. 点击课表页"+"按钮
2. 填写课程信息
3. 周次类型选择"双周"
4. 开始周次：1，结束周次：16
5. 点击添加

**结果：** 该课程只在第2、4、6、8、10、12、14、16周显示

### 示例3：自定义上课时间

1. 进入课表设置
2. 找到"上课时间设置"
3. 点击"第1节"
4. 修改开始时间为"08:30"
5. 修改结束时间为"09:15"
6. 点击确定
7. 保存设置

**结果：** 第1节的时间显示更新为 08:30-09:15

## 技术要点

### 1. 单双周判断算法

```dart
// 奇数周判断
week % 2 == 1

// 偶数周判断
week % 2 == 0
```

### 2. 时间表不可变更新

```dart
CustomTimeTable updateSection(int section, String startTime, String endTime) {
  final newSections = sections.map((s) {
    if (s.section == section) {
      return SectionTime(section: section, startTime: startTime, endTime: endTime);
    }
    return s;
  }).toList();
  return CustomTimeTable(sections: newSections);
}
```

### 3. 时间与课程分离

- 课程不再存储具体时间
- 时间由时间表统一管理
- 显示时动态获取
