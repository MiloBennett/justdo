# 设置按钮位置调整 - 改动记录

## 改动说明

将课表设置按钮从主页右上角移动到课表页面的右上角，更符合用户操作习惯。

## 改动文件

### 1. `lib/screens/main_screen.dart`

**移除内容：**
- 移除浮动按钮 `FloatingActionButton`
- 移除浮动按钮位置设置 `floatingActionButtonLocation`

**修改内容：**
- 向 `ScheduleScreen` 传递 `onSettingsPressed` 回调函数

```dart
// 修改前
ScheduleScreen(settings: _settings),

// 修改后
ScheduleScreen(
  settings: _settings,
  onSettingsPressed: _openSettings,
),
```

### 2. `lib/screens/schedule_screen.dart`

**新增参数：**
- 添加 `onSettingsPressed` 可选回调参数

```dart
class ScheduleScreen extends StatefulWidget {
  final ScheduleSettings settings;
  final VoidCallback? onSettingsPressed;

  const ScheduleScreen({
    super.key,
    required this.settings,
    this.onSettingsPressed,
  });
```

**新增按钮：**
- 在 AppBar actions 中添加设置图标按钮（位于日历按钮之前）

```dart
actions: [
  IconButton(
    icon: const Icon(Icons.settings),
    tooltip: '课表设置',
    onPressed: widget.onSettingsPressed,
  ),
  IconButton(
    icon: const Icon(Icons.calendar_today),
    // ... 原有代码
  ),
],
```

## 界面变化

| 位置 | 修改前 | 修改后 |
|------|--------|--------|
| 主页右上角 | 设置浮动按钮 | 无 |
| 课表页右上角 | 仅日历按钮 | 设置按钮 + 日历按钮 |

## 操作流程

1. 切换到"课表"页面
2. 点击右上角齿轮图标
3. 进入设置页面
4. 调整设置后保存
5. 返回课表页面查看效果
