# 待办页显示计划和课程 - 功能增加文档

## 功能概述

修改待办页面，使其不仅显示待办事项，还显示即将开始的日历计划和今日课程，方便用户快速了解当天和近期的安排。

## 修改文件

| 文件 | 修改类型 |
|------|----------|
| `lib/screens/main_screen.dart` | 修改 |
| `lib/screens/home_screen.dart` | 重写 |
| `lib/screens/calendar_screen.dart` | 修改 |
| `lib/screens/schedule_screen.dart` | 修改 |

## 详细修改说明

### 1. MainScreen (`lib/screens/main_screen.dart`)

**新增内容：**
- 管理共享数据（课程列表、日历计划列表）
- 提供数据增删方法供子页面调用

```dart
// 共享数据
final List<Course> _courses = [];
final List<Event> _events = [];

// 数据操作方法
void _addCourse(Course course);
void _removeCourse(Course course);
void _addEvent(Event event);
void _removeEvent(Event event);
void _toggleEventComplete(Event event);
```

### 2. HomeScreen (`lib/screens/home_screen.dart`)

**新增参数：**
```dart
final List<Course> courses;
final List<Event> events;
final int currentWeek;
final Function(Event) onEventComplete;
```

**新增功能模块：**

#### 今日课程卡片
- 显示今天的所有课程
- 按上课时间排序
- 显示课程名称、节次、教室、教师
- 左侧彩色边框标识课程颜色

#### 今日计划卡片
- 显示今天未完成的日历计划
- 按时间排序
- 显示计划标题、时间、地点
- 点击可标记完成

#### 近期计划卡片
- 显示最近7天的计划
- 最多显示5条
- 显示日期和星期

#### 待办事项
- 原有待办事项功能保留
- 位于页面底部

### 3. CalendarScreen (`lib/screens/calendar_screen.dart`)

**修改内容：**
- 使用父组件传递的数据替代本地数据
- 通过回调函数通知父组件数据变化

```dart
final List<Event> events;
final Function(Event) onAddEvent;
final Function(Event) onRemoveEvent;
final Function(Event) onToggleComplete;
```

### 4. ScheduleScreen (`lib/screens/schedule_screen.dart`)

**修改内容：**
- 使用父组件传递的数据替代本地数据
- 通过回调函数通知父组件数据变化

```dart
final List<Course> courses;
final Function(Course) onAddCourse;
final Function(Course) onRemoveCourse;
```

### 卡片样式

**课程卡片：**
- 左侧彩色边框（课程颜色）
- 课程名称（加粗）
- 节次信息
- 教室信息
- 教师信息
- 右上角显示星期

**计划卡片：**
- 左侧彩色边框（计划颜色）
- 计划标题（加粗）
- 时间范围
- 日期（非今日显示）
- 地点信息
- 右侧完成状态图标

## 数据流

```
MainScreen (数据管理中心)
    │
    ├── HomeScreen (读取数据)
    │   ├── 显示今日课程
    │   ├── 显示今日计划
    │   └── 显示近期计划
    │
    ├── ScheduleScreen (读写数据)
    │   ├── 读取课程数据
    │   └── 通过回调添加/删除课程
    │
    └── CalendarScreen (读写数据)
        ├── 读取计划数据
        └── 通过回调添加/删除/完成计划
```

## 使用说明

### 查看今日安排

1. 打开应用，默认进入待办页面
2. 页面顶部显示今日课程
3. 中部显示今日计划
4. 底部显示待办事项

### 标记计划完成

1. 在待办页面点击计划卡片
2. 计划标记为完成，显示删除线和勾选图标

### 添加待办事项

1. 点击右下角"+"按钮
2. 填写待办信息
3. 点击保存

### 添加课程/计划

1. 切换到课表/日历页面
2. 添加课程或计划
3. 返回待办页面可看到新增内容
