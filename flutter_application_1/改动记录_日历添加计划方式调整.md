# 日历页添加计划方式调整 - 修改记录

## 修改说明

将日历页的添加计划按钮取消，改为双击日期打开添加计划弹窗，操作更直观便捷。

## 修改文件

### `lib/screens/calendar_screen.dart`

**移除内容：**
- 移除浮动按钮 `FloatingActionButton`

```dart
// 移除前
floatingActionButton: FloatingActionButton(
  onPressed: _addEvent,
  child: const Icon(Icons.add),
),

// 移除后
// 无浮动按钮
```

**新增内容：**
- 日期添加双击事件 `onDoubleTap`

```dart
GestureDetector(
  onTap: () {
    setState(() {
      _selectedDate = date;
    });
  },
  onDoubleTap: () {
    setState(() {
      _selectedDate = date;
    });
    _addEvent();
  },
  child: Container(...)
)
```

**修改内容：**
- 提示文字从"点击右下角按钮添加计划"改为"双击日期添加计划"

```dart
// 修改前
Text('点击右下角按钮添加计划')

// 修改后
Text('双击日期添加计划')
```

## 操作方式变化

| 操作 | 修改前 | 修改后 |
|------|--------|--------|
| 选择日期 | 单击日期 | 单击日期 |
| 添加计划 | 点击右下角"+"按钮 | 双击日期 |

## 交互说明

1. **单击日期** - 选中日期，下方显示该日期的计划列表
2. **双击日期** - 选中日期并打开添加计划弹窗

## 界面变化

- 右下角浮动按钮已移除
- 无计划时提示文字更新为"双击日期添加计划"
