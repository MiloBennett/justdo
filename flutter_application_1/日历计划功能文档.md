# 日历计划功能实现文档

## 功能概述

新增日历计划页面，用于安排具体到时间的计划，支持月视图日历显示、计划时间安排、完成状态标记等功能。

## 新增文件

### 1. 数据模型

**文件**: `lib/models/event.dart`

计划数据模型包含以下字段：

| 字段 | 类型 | 说明 |
|------|------|------|
| `id` | String | 唯一标识 |
| `title` | String | 计划标题 |
| `description` | String | 计划描述 |
| `date` | DateTime | 计划日期 |
| `startTime` | TimeOfDay | 开始时间 |
| `endTime` | TimeOfDay | 结束时间 |
| `location` | String | 地点 |
| `color` | Color | 标记颜色 |
| `isCompleted` | bool | 完成状态 |

**提供的方法：**
- `dateString` - 获取日期字符串
- `timeRangeString` - 获取时间范围字符串
- `weekDayName` - 获取星期几名称
- `isToday` - 判断是否是今天
- `durationMinutes` - 获取持续时间（分钟）
- `isSameDate(DateTime)` - 判断是否与指定日期相同

### 2. 日历页面

**文件**: `lib/screens/calendar_screen.dart`

日历页面包含以下功能区域：

#### 月份导航栏
- 显示当前年月
- 左右箭头切换月份
- "今天"按钮快速返回当前日期

#### 日历网格
- 显示当前月份所有日期
- 周末日期显示为红色
- 当天日期高亮显示
- 选中日期特殊标记
- 有计划的日期显示彩色圆点

#### 计划列表
- 显示选中日期的所有计划
- 按时间顺序排列
- 显示时间、标题、地点、描述
- 支持标记完成状态
- 长按删除计划

### 3. 添加计划页面

**文件**: `lib/screens/add_event_screen.dart`

添加计划页面包含以下表单项：

| 表单项 | 说明 |
|--------|------|
| 计划标题 | 必填，计划名称 |
| 描述 | 可选，详细说明 |
| 地点 | 可选，计划地点 |
| 日期 | 选择计划日期 |
| 开始时间 | 选择开始时间 |
| 结束时间 | 选择结束时间 |
| 颜色 | 选择标记颜色 |

## 功能特点

### 日历视图

1. **月视图显示**
   - 完整显示一个月的日期
   - 自动排列星期一到星期日
   - 支持前后月份切换

2. **日期标记**
   - 当天高亮显示
   - 选中日期特殊样式
   - 周末红色显示
   - 有计划日期显示圆点

3. **快速导航**
   - 左右箭头切换月份
   - "今天"按钮快速返回

### 计划管理

1. **添加计划**
   - 选择日期后自动填充
   - 时间选择器精确到分钟
   - 8种颜色可选
   - 支持标题、描述、地点

2. **查看计划**
   - 按时间顺序显示
   - 显示时间范围
   - 显示地点和描述
   - 彩色左边框区分

3. **计划操作**
   - 点击标记完成/未完成
   - 长按删除计划
   - 完成后显示删除线

### 时间安排

1. **精确时间**
   - 时间选择精确到分钟
   - 显示开始和结束时间
   - 自动计算持续时间

2. **时间验证**
   - 结束时间必须晚于开始时间
   - 自动调整不合理时间

## 使用说明

### 进入日历

1. 点击底部导航栏"日历"标签
2. 进入日历计划页面

### 切换月份

1. 点击左箭头查看上个月
2. 点击右箭头查看下个月
3. 点击"今天"返回当前月份

### 选择日期

1. 点击日历中的任意日期
2. 选中日期高亮显示
3. 下方显示该日期的计划

### 添加计划

1. 点击右下角"+"按钮
2. 填写计划信息：
   - 输入标题
   - 输入描述（可选）
   - 输入地点（可选）
   - 选择日期
   - 选择开始时间
   - 选择结束时间
   - 选择颜色
3. 点击"添加计划"保存

### 管理计划

1. **标记完成**
   - 点击计划卡片
   - 完成状态切换
   - 完成后显示删除线和勾选图标

2. **删除计划**
   - 长按计划卡片
   - 确认删除对话框
   - 点击确认删除

## 技术实现

### 日历算法

```dart
// 获取月份第一天
final firstDayOfMonth = DateTime(year, month, 1);

// 获取月份天数
final lastDayOfMonth = DateTime(year, month + 1, 0);
final daysInMonth = lastDayOfMonth.day;

// 获取第一天是星期几
final firstWeekday = firstDayOfMonth.weekday;
```

### 日期比较

```dart
bool isSameDate(DateTime other) {
  return date.year == other.year && 
         date.month == other.month && 
         date.day == other.day;
}
```

### 时间排序

```dart
_events.sort((a, b) {
  final dateComparison = a.date.compareTo(b.date);
  if (dateComparison != 0) return dateComparison;
  final aMinutes = a.startTime.hour * 60 + a.startTime.minute;
  final bMinutes = b.startTime.hour * 60 + b.startTime.minute;
  return aMinutes.compareTo(bMinutes);
});
```
