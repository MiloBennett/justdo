# 日期时间选择器中文化 - 修改记录

## 修改说明

将添加计划页面的日期选择器和时间选择器的按钮文字从英文改为中文。

## 修改文件

### `lib/screens/add_event_screen.dart`

**日期选择器修改：**

```dart
// 修改前
final picked = await showDatePicker(
  context: context,
  initialDate: _selectedDate,
  firstDate: DateTime(2020),
  lastDate: DateTime(2030),
);

// 修改后
final picked = await showDatePicker(
  context: context,
  initialDate: _selectedDate,
  firstDate: DateTime(2020),
  lastDate: DateTime(2030),
  helpText: '选择日期',
  cancelText: '取消',
  confirmText: '确定',
);
```

**开始时间选择器修改：**

```dart
// 修改前
final picked = await showTimePicker(
  context: context,
  initialTime: _startTime,
);

// 修改后
final picked = await showTimePicker(
  context: context,
  initialTime: _startTime,
  helpText: '选择开始时间',
  cancelText: '取消',
  confirmText: '确定',
);
```

**结束时间选择器修改：**

```dart
// 修改前
final picked = await showTimePicker(
  context: context,
  initialTime: _endTime,
);

// 修改后
final picked = await showTimePicker(
  context: context,
  initialTime: _endTime,
  helpText: '选择结束时间',
  cancelText: '取消',
  confirmText: '确定',
);
```

## 界面变化

| 元素 | 修改前 | 修改后 |
|------|--------|--------|
| 日期选择器标题 | SELECT DATE | 选择日期 |
| 时间选择器标题 | SELECT TIME | 选择开始时间 / 选择结束时间 |
| 取消按钮 | CANCEL | 取消 |
| 确定按钮 | OK | 确定 |
